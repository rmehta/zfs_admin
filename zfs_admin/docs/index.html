Web Admin and User Portal for ZFS

Administer ZFS filesystem and share files via portal

Install

From your site

To install this app, login to your site and click on "Installer". Search for ZFS Admin and click on "Install"

Using Bench

Go to your bench folder and setup the new app

$ bench get-app zfs_admin https://github.com/rmehta/zfs_admin
$ bench new-site testsite
$ bench --site testsite install-app zfs_admin

Login to your site to configure the app.
